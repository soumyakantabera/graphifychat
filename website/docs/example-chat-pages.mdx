# Example chat pages

Use these examples as templates for scenario-focused product pages.

## Coding workflow page

- User shares bug context and affected file
- graphifychat stores decisions in TRON + Sparse6 layers
- Follow-up session resumes with compact memory file

## Research workflow page

- User explores topic with attachments and references
- graphifychat tracks intent, emotions, and open threads
- New chat starts with preserved context and clear priorities

## Creative workflow page

- User iterates ideas across many turns
- graphifychat captures concept links and direction shifts
- Story/brand decisions remain traceable between tools

## Recommended layout

- Header: scenario goal
- Middle: sample conversation or timeline
- Footer: quick actions (copy template, resume chat, read docs)
