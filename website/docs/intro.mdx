# graphifychat site overview

This static site is built with **React + Docusaurus** to present graphifychat in a clean, multi-page experience.

## What this site includes

- A homepage with tile-based navigation
- A usability-focused page for UX principles
- An example chat page to demonstrate practical usage patterns
- A reusable tile design specification page

## Local development

```bash
cd website
npm install
npm run start
```

## Production build

```bash
cd website
npm run build
```

The static output is generated in `website/build/`.
