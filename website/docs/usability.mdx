# Usability design

This page documents UX goals for the graphifychat website.

## UX principles

1. **Fast orientation** — users understand the product in under 10 seconds.
2. **Progressive disclosure** — simple overview first, detail on demand.
3. **Accessible hierarchy** — headings, spacing, and contrast are consistent.
4. **Actionable navigation** — every section links directly to useful next steps.

## Information architecture

- **Overview**: what graphifychat is and why it matters
- **Usability**: design principles and interaction behavior
- **Example chats**: concrete usage scenarios
- **UI tiles**: reusable design blocks for landing sections

## Interaction standards

- Keep sections short and scannable
- Prefer cards/tiles for top-level choices
- Use clear labels rather than abstract names
- Preserve responsive behavior on mobile first
